Pilgrimage (For the Internet)


HTML & CSS | PHP | Google Maps API | Javascript | Illustrator
December 2016
Nicole Lin & Eric Morgan

An interactive web and geo-based narrative experience. Pilgrimage takes the user on a walkabout around the city of Montreal, while discovering the new "religion" of the internet. Created with Javascript, integrating the Google Maps API.
