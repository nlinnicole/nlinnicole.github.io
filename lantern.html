Moth Lantern

March 2017
Blender & Substance Painter

This project was created for a 3D Modelling class where we were to design and model an original hero prop. I wanted to experiment with the sculpting feature in Blender, so I designed a more organically-shaped prop. I also wanted to experiment with Substance Painter for a more realistic result.
